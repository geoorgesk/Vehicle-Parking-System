# 🚗 MITS Vehicle Parking Management System

A full-stack web application developed to efficiently manage vehicle parking operations using **Flask, SQLite, Bootstrap, and Jinja2**.

This system supports both **User** and **Admin** functionalities including parking lot management, real-time spot booking, automatic billing, occupancy tracking, and analytics dashboard.

## 🚀 Features

### 👤 User Functionalities

- User Registration & Secure Login
- View Available Parking Lots
- Search Lots by Location, Address, or PIN
- Book Available Parking Spot
- Release Spot with Automatic Cost Calculation
- View Complete Parking History
- Edit User Profile
- View Parking Summary Dashboard

### 🛠️ Admin Functionalities

- Secure Admin Login
- Add / Edit / Delete Parking Lots
- Automatically Generate Multiple Parking Spots per Lot
- View Lot Details & Spot Status
- Manage Users
- Delete Users and Release Reserved Spots
- Revenue & Occupancy Summary Dashboard
- View Occupied Spot Details with Estimated Cost
